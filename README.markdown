# always_at_bottom

## Introduction

 If the content of web page is tiny then footer shows up in the middle of the page.
 This plugin ensures that footer is always at the bottom.

## depends on 

jQuery 1.3+

## How to use it
<pre>
	<code>
 		$('#footer').always_at_bottom();
	</code>
</pre>

## Tested with

Firefox, Safari, IE6, IE7
